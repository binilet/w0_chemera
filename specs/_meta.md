# Project Chimera — Meta Specification

## Vision
Project Chimera is an autonomous influencer AI system that researches trends, generates content, validates safety, and publishes posts with governance and human oversight.

## Core Constraints
- MUST follow specs before generating code
- MUST apply Planner–Worker–Judge agent pattern
- MUST require human approval for sensitive content
- MUST log all actions for traceability
- MUST prioritize safety over creativity

## Constraints
- Human-in-the-Loop approval required before publishing
- Must comply with platform policies (YouTube, TikTok, X)
- Must log all agent decisions (auditability)
-No irreversible actions without review
